Dùng để verify chữ ký HMAC-SHA256 trong header X-PKT-Signature.

Verify Signature

Header: X-PKT-Signature
Algo: HMAC-SHA256(secret, body)
Node.js example:
const sig = req.headers['x-pkt-signature'];
const expected = crypto
  .createHmac('sha256', SECRET)
  .update(rawBody)
  .digest('hex');
if (sig !== expected) return res.status(401).end();
