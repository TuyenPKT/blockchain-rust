Quickstart
1

Gọi REST API không cần auth

Tất cả endpoint GET /api/testnet/* đều public — không cần API key.

bash
# Lấy block mới nhất curl https://oceif.com/blockchain-rust/api/testnet/headers?limit=1 # Balance của một address curl https://oceif.com/blockchain-rust/api/testnet/balance/pkt1q...
2

Gọi ETH JSON-RPC EVM

Endpoint POST /eth — tương thích hoàn toàn với Ethereum JSON-RPC 2.0.

bash — eth_blockNumber
curl -X POST https://oceif.com/blockchain-rust/eth \ -H "Content-Type: application/json" \ -d '{"jsonrpc":"2.0","method":"eth_blockNumber","params":[],"id":1}'
bash — eth_getBalance
curl -X POST https://oceif.com/blockchain-rust/eth \ -H "Content-Type: application/json" \ -d '{"jsonrpc":"2.0","method":"eth_getBalance","params":["0xAddress","latest"],"id":1}'
bash — eth_call (smart contract read)
curl -X POST https://oceif.com/blockchain-rust/eth \ -H "Content-Type: application/json" \ -d '{ "jsonrpc": "2.0", "method": "eth_call", "params": [{"to":"0xContract","data":"0x70a08231..."},"latest"], "id": 1 }'
3

Cài JS/TS SDK

SDK typed đầy đủ, hỗ trợ REST + JSON-RPC + WebSocket.

bash
npm install @pktcore/sdk
typescript
import { PktClient } from '@pktcore/sdk'; const client = new PktClient({ baseUrl: 'https://oceif.com/blockchain-rust', }); const summary = await client.getSummary(); console.log(summary.height, summary.hashrate); // EVM — eth_call via JSON-RPC const bal = await client.eth.getBalance('0xAddress'); console.log('Balance (wei):', bal);
4

Subscribe real-time qua WebSocket

Nhận block/tx mới không cần polling.

typescript
client.subscribe('block', (block) => { console.log('New block:', block.height, block.hash); }); client.subscribe('tx', (tx) => { console.log('New tx:', tx.tx_id, tx.fee); });
5

Đăng ký Webhook (cần write API key)

Nhận HTTP POST khi có event — không cần giữ WebSocket connection.

bash
curl -X POST https://oceif.com/blockchain-rust/api/webhooks \ -H "X-API-Key: your_write_key" \ -H "Content-Type: application/json" \ -d '{"url":"https://your-app.com/hook","events":["new_block"]}'

Payment channels Layer 2 trên PKT — instant, low-fee transactions.

open_channel Fund + mở payment channel

HTLC script — OP_IF hash-path / OP_ELSE timelock-path
// Offered HTLC: receiver dùng preimage, sender dùng CLTV expiry let script = Script::htlc_offered( &payment_hash, // SHA256 của preimage &receiver_pkh, // beneficiary pubkey hash &sender_pkh, // refund pubkey hash expiry_blockheight, ); // Spend path 1: receiver cung cấp preimage let ctx = SpendContext { lock_time: 0, sequence: 0 }; let ok = script.execute_with_context(&preimage_witness, &sig, &ctx); // Spend path 2: sender refund sau expiry let ctx = SpendContext { lock_time: expiry_blockheight + 1, sequence: 0xFFFFFFFF }; let ok = script.execute_with_context(&[], &sender_sig, &ctx);

Verify Webhook Signature

Mỗi request từ PKTScan có header X-PKT-Signature = HMAC-SHA256(secret, body). Verify để đảm bảo request đến từ PKTScan.

Node.js / Express
import crypto from 'crypto'; app.post('/hook', (req, res) => { const sig = req.headers['x-pkt-signature']; const expected = crypto .createHmac('sha256', process.env.PKT_WEBHOOK_SECRET) .update(req.rawBody) .digest('hex'); if (sig !== expected) return res.status(401).end(); const { event, data } = req.body; if (event === 'new_block') console.log('Block:', data.height); res.sendStatus(200); });
